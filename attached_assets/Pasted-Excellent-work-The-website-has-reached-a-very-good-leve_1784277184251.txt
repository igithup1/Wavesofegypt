====================================================

Instead of a traditional shopping cart, implement a premium "My Trip" feature.

- Add a "My Trip" button in the header showing the number of selected tours.
- Save selections even after page refresh.

Create a beautiful "My Trip" page including:

- Selected tours
- Tour thumbnail
- Tour name
- Starting price
